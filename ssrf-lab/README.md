# ssrf-labs

## Requirements

```
▶ Flask==0.10.1
▶ requests==2.19.1
▶ tldextract==3.1.2
```

## Installing Dependencies


```
▶ python3 -m pip install -r requirements.txt

```

## Running SSRF Application

```
▶ git clone https://github.com/spiderSilk-com/blogs
▶ cd blogs/ssrf-labs
▶  python3 -m pip install -r requirements.txt
▶ python3 ssrf.py
```
